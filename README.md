# 🚀 Projectile Motion Simulator
A simple Python program that calculates the motion of a projectile launched from ground level.

## Features

- Calculates flight time
- Calculates maximum height
- Calculates horizontal range
- User-friendly input
- Uses standard projectile motion equations

## Physics

The program assumes:

- No air resistance
- Constant gravity (9.81 m/s²)
- Launch from ground level

## Technologies

- Python
- math module

## Example

```
Initial speed: 20
Launch angle: 45

Flight time: 2.883 s
Maximum height: 10.194 m
Range: 40.775 m
```
